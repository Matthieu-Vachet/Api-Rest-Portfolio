## Pre-requis

-   Node.js v21.6.1
-   Npm 10.4.0

## Check Node.js version

`node --version`

## Check Mongo version

`mongo --version`

## Install dependencies

-   `yarn install` or `npm install`

## Start server

-   `yarn start` or `npm start`
